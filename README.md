# ClassVerter 0.1
Javascript library to convert from one class language to another

## Language available:
- C#
- Java
- Javascript
- Typescript
- PHP

## And format...:
- Json
- JsonSchema

## Api documentation:

```c
	var converter = new Converter();
    converter.convertToClass(jsonText, typeFrom, typeTo, { isProperty: isProperty, className: jsonClassName, doc: doc });
```
